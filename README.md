# eveFace -暂时未更新-
一个基于leetaface的人脸识别项目</br>
## 介绍:
该项目使用了山世光老师的开源项目leetaface并进行了多线程设计，该库提供了特征提取和特征匹配的接口，</br>
也提供了人脸检测的接口，不过我使用另外一个人脸检测库进行了替换。建立该repository的目的一方面是方</br>
便我对自己的项目进行管理，另一方面是供有兴趣的朋友进行Diy设计。</br>
